# Discord Music Bot

## General
 Discord bot made with Discord.js and Distube modules.

# Config.js

You must create "Config.js" to your application work.

```
{
    "prefix": "your-prefix-here",
    "token": "your-token-here",
    "mongoPath": "your-mongo-link-here"
}
```
